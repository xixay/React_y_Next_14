## fylter
- Este código utiliza filter para crear un nuevo array que contiene solo los elementos que cumplen con una condición específica.
```js
//importar la libreria
import fetch from 'node-fetch';

//async await
const obtenerPokemones = async () => {
  try {
    const res = await fetch('https://pokeapi.co/api/v2/pokemon/');
    const data = await res.json();
    const arrayNombres = data.results.filter(poke => poke.name === 'bulbasaur');
    console.log(arrayNombres);
  } catch (error) {
    console.log(error);
  }
};

obtenerPokemones();
```
- respuesta nuevo array filtrado
```js
[ { name: 'bulbasaur', url: 'https://pokeapi.co/api/v2/pokemon/1/' } ]
```
