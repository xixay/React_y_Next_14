## 1. Objetos en JavaScript con anidamiento y desestructuración
- En JavaScript, los objetos pueden contener otros objetos dentro de ellos. Esto es útil para organizar datos relacionados. A continuación, vemos un ejemplo de un objeto anidado con diferentes niveles de propiedades.
- En este ejemplo, el objeto web contiene propiedades como nombre, links y redesSociales, y dentro de redesSociales, tenemos un objeto youtube con su propio conjunto de propiedades.
```js
const web = {
  nombre: 'bluwweb',
  links: {
    enlace: 'www.bluuweb.cl',
  },
  redesSociales: {
    youtube: {
      enlace: 'youtube.com/bluuweb',
      nombre: 'bluuweb yt',
    },
  },
};
console.log(web);
```
- respuesta
```js
{
  nombre: 'bluwweb',
  links: {
    enlace: 'www.bluuweb.cl'
    },
  redesSociales: {
    youtube: {
      enlace: 'youtube.com/bluuweb',
      nombre: 'bluuweb yt'
      }
    }
}
```
## 2. Acceder a propiedades anidadas:
- Si deseas acceder a propiedades dentro de un objeto anidado, puedes hacerlo utilizando la notación de punto. Por ejemplo, para obtener el enlace de YouTube:
- En este caso, estamos accediendo al enlace de YouTube dentro del objeto redesSociales, que a su vez está dentro de web.
```js
const enlaceYT = web.redesSociales.youtube.enlace;
console.log(enlaceYT); //youtube.com/bluuweb
```
## 3. Desestructuración de objetos anidados:
- La desestructuración es una forma conveniente de extraer propiedades específicas de un objeto anidado de manera directa.
```js
const { enlace, nombre } = web.redesSociales.youtube;
console.log(enlace);// 'youtube.com/bluuweb'
console.log(nombre);// 'bluuweb yt'
```
