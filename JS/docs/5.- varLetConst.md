## 1. var
```js
var edad = 10;
var edad = 20;
console.log(edad); //20
```
## 2. let
```js
let edad = 10;
let edad = 20;
console.log(edad); //error

//solucion
let edad = 10;
edad = 20;
console.log(edad);

var edad = 10;
if (true) {
  var edad = 20;
  console.log(edad); //20
}
console.log(edad); //20
```
## 3. constantes
```js
//constantes
const edad = 10;
if (true) {
  const edad = 20;
  console.log(edad); //20
}
console.log(edad); //10
```
## 4. Modificacion de array ctes y objetos
- ctes array o objetos pueden ser modificadas
```js
const arrayNumero = [10, 20, 30];
arrayNumero.push(40);
console.log(arrayNumero); //[10,20,30,40] aunque sea cte agrega el 40

const persona = {
  nombre: 'Juanito',
  edad: 20,
};
persona.edad = 21;
persona.pais = 'Mexico';
console.log(persona);
```
