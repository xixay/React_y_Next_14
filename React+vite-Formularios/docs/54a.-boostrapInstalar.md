## Instalar
- Instalar Bootstrap
```
npm install bootstrap
```
## Importar Estilos de PrimeReact
- Si quieres centralizar los estilos en un archivo CSS, abre tu archivo index.css o App.css y agrega lo siguiente:
```css
/* index.css o App.css */
@import 'bootstrap/dist/css/bootstrap.min.css';
```
- Para ver si funciono en el archivo App.jsx, importar el boton de booststrap
```jsx
import Nocontrolado from './components/NoControlado';
import { Button } from 'primereact/button';

const App = () => {
  return (
    <>
      <Nocontrolado />
      <button className="btn btn-primary">Click Me</button>
    </>
  );
};
export default App;
```
