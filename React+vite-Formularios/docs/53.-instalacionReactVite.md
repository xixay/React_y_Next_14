# Instalación y Configuración Inicial
## Instalación del Proyecto
### Crear Proyecto
- Ejecuta el comando para iniciar un nuevo proyecto en la carpeta actual:
```txt
npm create vite@latest .
```
- Si aparece "Ok to proceed?", presiona y.
```txt
Ok to proceed? (y)
```
### Requisitos de Node.js:
- Asegúrate de tener Node.js versión 16 o superior.
- Pedira Condiciones crear el proyecto
**![Normal](/React+vite-Formularios/imagenes/53.-instalacionReactVite.png)**

### Instalar Dependencias:
- Una vez creado el proyecto, instala los paquetes necesarios:
```
npm i
```
### Ejecutar el Proyecto
- Luego hacer correr
**![Normal](/React+vite-Formularios/imagenes/53.-instalacionReactViteEjec.png)**

## Limpieza del Proyecto
### Vaciar index.css:
- Limpia el archivo index.css para empezar desde cero.
**![Normal](/React+vite-Fundamentos/imagenes/23.-indexCss.png)**

### Eliminar Archivos Innecesarios:
- Borra App.css, App.jsx y cualquier imagen dentro de la carpeta assets.
**![Normal](/React+vite-Fundamentos/imagenes/23.-eliminar.png)**

**![Normal](/React+vite-Fundamentos/imagenes/23.-eliminarImg.png)**

### Configurar App.jsx:
- Reemplaza su contenido con:
```jsx
const App = () => {
  return 'App';
};
export default App;
```
**![Normal](/React+vite-Formularios/imagenes/53.-instalacionReactViteEjechost.png)**

## Verificación
- Si tienes instalada la extensión React Developer Tools, podrás inspeccionar el proyecto directamente desde el navegador.
**![Normal](/React+vite-Formularios/imagenes/53.-instalacionReactViteEjecComp.png)**
