## Instalar
- En la terminal, al agregar el punto crea el proyecto directamente en la carpeta formularios-react
```txt
npm create vite@latest .
```
- Las primeras veces sale
```txt
Ok to proceed? (y)
```
- Te pedira que instales una version de node 16 o >=20
- Pedira Condiciones crear el proyecto
**![Normal](/React+vite-Formularios/imagenes/53.-instalacionReactVite.png)**

- Luego instalar los paquetes
```
npm i
```
- Luego se puede hacer correr
**![Normal](/React+vite-Formularios/imagenes/53.-instalacionReactViteEjec.png)**
