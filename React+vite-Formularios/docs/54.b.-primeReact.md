# primeReact
## Instalar
- Instalar PrimeReact y Dependencias
```
npm install primereact primeicons
```
## Importar Estilos de PrimeReact
- PrimeReact requiere que importes sus estilos globales en tu proyecto. Abre tu archivo principal de estilos, generalmente index.css o App.css, y agrega las siguientes líneas:
```css
/* Core CSS de PrimeReact */
@import 'primereact/resources/themes/lara-light-blue/theme.css';
@import 'primereact/resources/primereact.min.css';

/* Iconos */
@import 'primeicons/primeicons.css';
```
- Para ver si funciono en el archivo App.jsx, importar el boton de react prime
```jsx
import Nocontrolado from './components/NoControlado';
import { Button } from 'primereact/button';

const App = () => {
  return (
    <>
      <Nocontrolado />
      <Button label="Click Me" icon="pi pi-check" />
    </>
  );
};
export default App;
```
