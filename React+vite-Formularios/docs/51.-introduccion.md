## Planificacion
### Formularios no controlados
- El DOM maneja los valores de los inputs directamente. React no los gestiona.
### Formularios controlados
- React controla y sincroniza los valores de los inputs mediante el estado.
## Recursos
### Form React:
- Crear formularios interactivos con React.
### Uncontrolled Components
- Inputs no gestionados por React. Útil para simplicidad o integraciones externas.
### useRef
- Referencia para manipular elementos del DOM o valores sin re-renderizar.
### FormData
- API para trabajar con datos de formularios en JavaScript.
