## Preparacion
- Crear la carperta components
- Crear adentro el archivo NoControlado.jsx
```jsx
const Nocontrolado = () => {
  return 'NoControlado';
};
export default Nocontrolado;
```
- Luego en el archivo App.jsx, importar el componente
```jsx
import Nocontrolado from './components/NoControlado';

const App = () => {
  return (
    <>
      <Nocontrolado />
    </>
  );
};
export default App;
```
