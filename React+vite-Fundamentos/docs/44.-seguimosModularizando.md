# Seguimos Modularizando
## Crear el Componente WelcomeText
- Definición: Crea un componente funcional que muestra "Online" u "Offline" basado en la prop user.
```jsx
//Desestructurando y por devolver solo un jsx en una linea se omite el return y las llaves
const WelcomeText = ({user}) => user ? <h3>Online</h3>: <h3>Offline</h3>

export default WelcomeText
```
## Optimizar el Componente WelcomeText
- Definición: Simplifica el componente utilizando un operador ternario dentro de una sola etiqueta <h3> para mayor eficiencia y claridad.
```jsx
//Desestructurando y por devolver solo un jsx en una linea se omite el return y las llaves
const WelcomeText = ({user}) => <h3>{user?'Online':'Offline'}</h3>

export default WelcomeText
```
## Importar WelcomeText en App.jsx
- Definición: Importa el componente WelcomeText desde la carpeta components para usarlo en tu aplicación principal.
```jsx
import WelcomeText from './components/WelcomeText'
```
## Crear el Componente ItemFrut
- Definición: Crea un componente funcional que recibe una fruta como prop y la renderiza dentro de un elemento <li>. Se elimina la prop index ya que no es necesaria dentro del componente.
```jsx
//Un componente que devuelve listas
const ItemFrut = ({frut}) => {
    return (
      <li>{frut}</li>
    )
  }
export default ItemFrut
```
## Importar ItemFrut en App.jsx
- Definición: Importa el componente ItemFrut desde la carpeta components para utilizarlo en la lista de frutas.
```jsx
import ItemFrut from './components/ItemFrut'
```
## Renderizar la Lista de Frutas
- Definición: Utiliza el método map para iterar sobre el array fruts y renderizar cada fruta usando el componente ItemFrut. Asegúrate de proporcionar una key única para cada elemento de la lista.
```jsx
<ul>
{
    fruts.map((frut, index) => (
    // <li key={index}>{frut} - {index}</li>//Sera remplazado
    <ItemFrut key = {index} frut = {frut}/>
    ))
}
</ul>
```
## Código Completo de App.jsx
- Definición: Integra todos los componentes importados (MyButton, WelcomeText, ItemFrut) en el componente principal App, manteniendo una estructura organizada y modular.
```jsx
import imgUno from './assets/images/img-1.jpg'

import MyButton from './components/MyButton'
import WelcomeText from './components/WelcomeText'
import ItemFrut from './components/ItemFrut'

const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  // const pathImg = 'src/assets/images/img-1.jpg';
  const user = false;

  const fruts =['manzana', 'platano', 'pera'];

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={imgUno} alt={`imagen-${title}`} />

      <MyButton text = 'botòn 1'/>
      <MyButton text = 'botòn 2'/>
      <MyButton text = 'botòn 3'/>
      <WelcomeText user = {user}/>
      <ul>
        {
          fruts.map((frut, index) => (
            // <li key={index}>{frut} - {index}</li>//Sera remplazado
            <ItemFrut key = {index} frut = {frut}/>
          ))
        }
      </ul>
    </>
  )
};

export default App;
```
