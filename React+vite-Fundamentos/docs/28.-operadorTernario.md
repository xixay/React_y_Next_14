# 1. Operador ternario
## 1.1. Renderizado condicional
- Es una forma de escribir un if-else en JavaScript de manera más compacta.
- En este ejemplo, como la constante user es true, se muestra el texto "Online".
```jsx
const MyButton = () => {
  return (
    <button>Soy un boton</button>
  )
 }

const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  const pathImg = 'https://picsum.photos/200/300';
  const user = true;

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={pathImg} alt={`imagen-${title}`} />
      <MyButton></MyButton>
      {/* Si user es verdadero, muestra "Online", de lo contrario, "Offline" */}
      {
        user ? 'Online' : 'Offline'
      }
    </>
  )
};

export default App;
```
