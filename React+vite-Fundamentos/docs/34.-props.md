## Props
- A los componentes en React se les pueden pasar props para personalizar su contenido o comportamiento.
- En este caso, al componente MyButton se le está pasando un prop llamado text.
- Este prop es un objeto que contiene una propiedad text, la cual se utiliza dentro del componente para renderizar el contenido del botón.
```jsx
import imgUno from './assets/images/img-1.jpg'

const MyButton = (props) => {
console.log(props)
  return (
    <button>{props.text}</button>
  )
 }

const OnlineText = () =>{
  return (
    <div>
      <h3>Online</h3>
    </div>
  )
}
const OfflineText = () =>{
  return (
    <div>
      <h3>Offline</h3>
    </div>
  )
}
const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  // const pathImg = 'src/assets/images/img-1.jpg';
  const user = false;

  const fruts =['manzana', 'platano', 'pera'];

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={imgUno} alt={`imagen-${title}`} />

      <MyButton text = 'botòn 1'/>
      <MyButton text = 'botòn 2'/>
      <MyButton text = 'botòn 3'/>
      {user && <OnlineText/>}
      <ul>
        {
          fruts.map((frut, index) => (
            <li key={index}>{frut} - {index}</li>
          ))
        }
      </ul>
    </>
  )
};

export default App;
```

### Explicación:
- props es el objeto que agrupa todos los valores pasados como atributos al componente.
- Dentro del componente, se puede acceder al valor del prop utilizando props.nombreDelProp.
- En el caso del componente MyButton, el texto del botón se establece dinámicamente mediante props.text.
