## Desactivar Eslint
- ESLint es una herramienta para identificar y reportar patrones en el código, ayudando a mantener buenas prácticas.
- Si necesitas desactivar ESLint en un archivo o fragmento de código, puedes usar comentarios especiales que instruyen a ESLint a omitir ciertas reglas.
### Para desactivar ESLint en una línea específica:
```js
// eslint-disable-next-line
console.log("Esta línea no será revisada por ESLint");
```
### Para desactivar ESLint en un bloque de código:
```js
/* eslint-disable */
console.log("Este bloque no será revisado por ESLint");
console.log("Se pueden añadir múltiples líneas aquí.");
/* eslint-enable */
```
### Si deseas desactivar una regla específica:
```js
// eslint-disable-next-line no-console
console.log("Esta línea usa `console.log`, que podría estar prohibido por una regla de ESLint");
```
## Nota
- Desactivar ESLint es útil durante el desarrollo rápido o para evitar conflictos temporales, pero es una práctica recomendada revisarlo y reactivarlo en el código final para mantener estándares de calidad.
