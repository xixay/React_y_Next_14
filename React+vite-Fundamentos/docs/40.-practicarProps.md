# 1. Practicando con props
## 1.1. ¿Qué se está practicando?
- Uso de props para hacer los componentes más reutilizables y dinámicos.
- Uso de desestructuración para simplificar el acceso a las propiedades.
- Incorporación de lógica condicional en componentes.

## 1.2. Ejm
- Volver dinamico el componente del mensaje
```jsx
import imgUno from './assets/images/img-1.jpg'

const MyButton = ({ text }) => {
  // console.log(props)
  return (
    <button>{text}</button>
  )
 }
// Componente con lógica condicional para mostrar el estado del usuario
const WelcomeText = ({user}) =>{
  if (user) {
    return (
      <h3>Online</h3>
    )
  }
  return (
    <h3>Offline</h3>
  )
}
// Componente para renderizar elementos de una lista
const ItemFrut = ({ index, frut }) => {
  return (
    <li key={index}>{frut} - {index}</li>
  )
}
const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  // const pathImg = 'src/assets/images/img-1.jpg';
  const user = false;

  const fruts =['manzana', 'platano', 'pera'];

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={imgUno} alt={`imagen-${title}`} />

      <MyButton text = 'botón 1'/>
      <MyButton text = 'botón 2'/>
      <MyButton text = 'botón 3'/>
      <WelcomeText user = {user}/>
      <ul>
        {
          fruts.map((frut, index) => (
            // <li key={index}>{frut} - {index}</li>//Sera remplazado
            <ItemFrut key = {index} frut = {frut}/>
          ))
        }
      </ul>
    </>
  )
};

export default App;
```
