### Volar
- Soporte completo para Vue en VS Code.
### ES7+ React/Redux/React-Native Snippets
- Atajos para escribir código React rápidamente.
### Material Icon Theme
- Íconos atractivos para organizar visualmente tus proyectos.
- ### Nu Disco Theme
- Tema colorido y moderno para el editor.
## Atajos para Funciones de Flecha
### anfn
- Genera una función de flecha anónima.
```js
(param) => { body }
```
### nfn
- Crea una función de flecha nombrada.
```js
const name = (param) => { body }
```
