## Instalar
- En la terminal
```txt
npm create vite@latest
```
- Las primeras veces sale
```txt
Ok to proceed? (y)
```
- Te pedira que instales una version de node 16 o >=20
- Pedira Condiciones crear el proyecto
**![Normal](/React/imagenes/21.-instalacionViteReact.png)**
- Luego ingresar a el proyecto creado
```
cd fundamentos-react
```
- Luego instalar los paquetes
```
npm i
```
