# 1. Vite
## 1.1. Crear un proyecto con Vite
- En la terminal
```txt
npm create vite@latest
```
## 1.2. Confirmar los pasos iniciales
- Durante la instalación, podría preguntar:
```txt
Ok to proceed? (y)
```
## 1.3. Verificar la versión de Node.js
- Si es necesario, actualiza a Node.js 16 o superior.
## 1.4. Configurar el proyecto
- El asistente pedirá opciones para configurar el proyecto:
**![Normal](/React+vite-Fundamentos/imagenes/21.-instalacionViteReact.png)**
## 1.5. Acceder al directorio del proyecto
- En la terminal
```
cd fundamentos-react
```
## 1.6. Instalar dependencias
- Luego instalar los paquetes
```
npm i
```
