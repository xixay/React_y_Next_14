## 1. PropTypes
- PropTypes es una librería de validación de propiedades que se usa para asegurar que los componentes de React reciban las propiedades adecuadas en el tipo y formato esperado.
- En el ejemplo proporcionado, MyButton recibe dos propiedades: text y edad. Estas propiedades son validadas utilizando PropTypes para asegurar que:
  - text es una cadena de texto (string) y es obligatorio (isRequired).
  - edad es un número (number) y también es obligatorio (isRequired).
### 1.1. Ejemplo básico de uso de PropTypes:
```jsx
import ProTypes from 'prop-types'
const MyButton = (props) => {
  return (
    <button>{props.text} - {props.edad}</button>
  )
 }

 MyButton.proTypes = {
  text: ProTypes.string.isRequired,//El campo sera string y sera requerido
  edad: ProTypes.number.isRequired,//El campo sera numerico y sera requerido
 }

const App = () => {
  return (
    <div>
      <MyButton text = 'text'></MyButton>
    </div>
  )
};
export default App;
```

## 2. ¿Por qué usar PropTypes?
- Validación de tipos: Asegura que las propiedades que se pasan a un componente son del tipo correcto (como string, number, array, etc.).
- Errores de desarrollo: Si un tipo de propiedad no coincide con el tipo esperado, React mostrará un error de consola, lo que ayuda a detectar problemas rápidamente.
- Mantenimiento del código: Facilita la lectura y el mantenimiento del código, ya que los tipos de las propiedades están claramente definidos.
## 3. Tipos de validaciones comunes en PropTypes:
- PropTypes.string para cadenas de texto.
- PropTypes.number para números.
- PropTypes.bool para valores booleanos.
- PropTypes.array para arreglos.
- PropTypes.object para objetos.
- PropTypes.func para funciones.
