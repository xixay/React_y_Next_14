## PropTypes
- para el ejm nuestro App.jsx, sera de la sgte forma
```jsx
const MyButton = (props) => {
  return (
    <button>{props.text}</button>
  )
 }

const App = () => {
  return (
    <div>
      <MyButton text = 'text'></MyButton>
    </div>
  )
};
export default App;
```
- ProTypes, son las propiedades que se le va pasar a nuestros componentes
- Importar los ProTypes
```jsx
import ProTypes from 'prop-types'
const MyButton = (props) => {
  return (
    <button>{props.text} - {props.edad}</button>
  )
 }

 MyButton.proTypes = {
  text: ProTypes.string.isRequired,//El campo sera string y sera requerido
  edad: ProTypes.number.isRequired,//El campo sera numerico y sera requerido
 }

const App = () => {
  return (
    <div>
      <MyButton text = 'text'></MyButton>
    </div>
  )
};
export default App;
```
