## Fragment
- Al no tener div estilos en App.jsx, se puede usar sin etiqueta
```jsx
const App = () => {
  return (
    <>
      <h1 className="text-center">Hola soy react</h1>
      <img src="https://picsum.photos/200/300" alt="" />
    </>
  )
};

export default App;

```
- Al usar fragment desaparece el div de que encerraba a h1 y img, funciona de la misma manera
**![Normal](/React+vite-Fundamentos/imagenes/25.-fragment.png)**

- fragment puede ser llamada desde la biblioteca de react, como una exportacion nombrada
```jsx
import { Fragment } from "react";
const App = () => {
  return (
    <Fragment>
      <h1 className="text-center">Hola soy react</h1>
      <img src="https://picsum.photos/200/300" alt="" />
    </Fragment>
  )
};
export default App;
```
- O el fragment puede ser llamado desde todo React
```jsx
import React from "react";
const App = () => {
  return (
    <React.Fragment>
      <h1 className="text-center">Hola soy react</h1>
      <img src="https://picsum.photos/200/300" alt="" />
    </React.Fragment>
  )
};
export default App;
```
