# 1. Operador '&&'
## 1.1. Renderizado Condicional Simplificado
- Utiliza el operador && para renderizar un componente solo si la condición es verdadera.
- Si user es true, se muestra el componente asociado. Si es false, no se renderiza nada.
## 1.2. Ejemplo con user = true:
```jsx
const MyButton = () => {
  return (
    <button>Soy un boton</button>
  )
 }

const OnlineText = () =>{
  return (
    <div>
      <h3>Online</h3>
    </div>
  )
}
const OfflineText = () =>{
  return (
    <div>
      <h3>Offline</h3>
    </div>
  )
}
const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  const pathImg = 'https://picsum.photos/200/300';
  const user = true;

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={pathImg} alt={`imagen-${title}`} />
      <MyButton></MyButton>
      {/* pregunta si usuario es verdadero */}
      {
        user && <OnlineText/>
      }
    </>
  )
};

export default App;
```
