# 1. Operador ternario
## 1.1. Renderizado condicional con componentes
- Se utiliza un componente para mostrar contenido diferente según el valor de user.
- En este ejemplo, se han creado dos componentes:
  - OnlineText: Se muestra cuando user es true.
  - OfflineText: Se muestra cuando user es false.
```jsx
const MyButton = () => {
  return (
    <button>Soy un boton</button>
  )
 }

const OnlineText = () =>{
  return (
    <div>
      <h3>Online</h3>
    </div>
  )
}
const OfflineText = () =>{
  return (
    <div>
      <h3>Offline</h3>
    </div>
  )
}
const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  const pathImg = 'https://picsum.photos/200/300';
  const user = true;

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={pathImg} alt={`imagen-${title}`} />
      <MyButton></MyButton>
      {/* pregunta si usuario es verdadero */}
      {
        user ? <OnlineText/> : <OfflineText/>
      }
    </>
  )
};

export default App;
```
