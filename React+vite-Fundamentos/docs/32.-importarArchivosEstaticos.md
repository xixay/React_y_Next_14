# Importar Archivos Estáticos en React con Vite
## Guardar la Imagen en el Proyecto
- Crea una carpeta llamada images dentro de src/assets y guarda la imagen como img-1.jpg.
## Intento con Rutas Relativas
- Se intentó usar una ruta relativa para referenciar la imagen, pero esto no funcionó correctamente en Vite.
```jsx
const pathImg = './assets/images/img-1.jpg'; // No se encuentra.
```
```jsx
const MyButton = () => {
  return (
    <button>Soy un boton</button>
  )
 }

const OnlineText = () =>{
  return (
    <div>
      <h3>Online</h3>
    </div>
  )
}
const OfflineText = () =>{
  return (
    <div>
      <h3>Offline</h3>
    </div>
  )
}
const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  const pathImg = './assets/images/img-1.jpg';
  const user = false;

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={pathImg} alt={`imagen-${title}`} />
      <MyButton></MyButton>
      {user && <OnlineText/>}
    </>
  )
};

export default App;
```

## Uso de Rutas Absolutas Relativas al Proyecto
- Una solución funcional es usar la ruta relativa desde src.
```jsx
const pathImg = 'src/assets/images/img-1.jpg'; // Funciona correctamente.
```

## Importación Directa de la Imagen (Recomendada)
- Vite permite importar imágenes como módulos en tu archivo JavaScript. Esto es más limpio y confiable.
```jsx
import imgUno from './assets/images/img-1.jpg';
```
```jsx
import imgUno from './assets/images/img-1.jpg'

const MyButton = () => {
  return (
    <button>Soy un boton</button>
  )
 }

const OnlineText = () =>{
  return (
    <div>
      <h3>Online</h3>
    </div>
  )
}
const OfflineText = () =>{
  return (
    <div>
      <h3>Offline</h3>
    </div>
  )
}
const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  // const pathImg = 'src/assets/images/img-1.jpg';
  const user = false;

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={imgUno} alt={`imagen-${title}`} />
      <MyButton></MyButton>
      {user && <OnlineText/>}
    </>
  )
};

export default App;
```
