# 1. Desestructuración
## 1.1. ¿Qué es la desestructuración en React?
- La desestructuración permite extraer propiedades de un objeto, como props, y asignarlas directamente a variables. Esto evita repetir props cada vez que accedemos a sus propiedades.
```jsx
import imgUno from './assets/images/img-1.jpg'

const MyButton = ({text}) => {
  // console.log(props)
  return (
    <button>{text}</button>
  )
 }

const OnlineText = () =>{
  return (
    <div>
      <h3>Online</h3>
    </div>
  )
}
const OfflineText = () =>{
  return (
    <div>
      <h3>Offline</h3>
    </div>
  )
}
const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  // const pathImg = 'src/assets/images/img-1.jpg';
  const user = false;

  const fruts =['manzana', 'platano', 'pera'];

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={imgUno} alt={`imagen-${title}`} />

      <MyButton text = 'botón 1'/>
      <MyButton text = 'botón 2'/>
      <MyButton text = 'botón 3'/>
      {user && <OnlineText/>}
      <ul>
        {
          fruts.map((frut, index) => (
            <li key={index}>{frut} - {index}</li>
          ))
        }
      </ul>
    </>
  )
};

export default App;
```
