# 1. jsx
## 1.1. ¿Qué es JSX?
- Es una combinación de JavaScript y XML.
- JSX traduce las etiquetas a código JavaScript mediante React.createElement, lo que permite mostrar información similar a HTML, pero no es HTML puro.
## 1.2. Estilizado en JSX
- Para definir estilos se usa className en lugar de class:
```jsx
//- para llevarlo a otro lugar
//- Las exportaciones por default solo se puede llevar una cosa
//- En el destino js sabe que vendra una sola cosa, por ello si se quiere se puede llamar de otra forma
const App = () => {
  return <h1 className="text-center">Hola soy react</h1>;
};

export default App
```
- Crea el estilo correspondiente en index.css:
```css
.text-center {
  text-align: center;
}
```

## 1.3. Retornar múltiples elementos
- Usa paréntesis y un contenedor, como div:
```jsx
//- para llevarlo a otro lugar
//- Las exportaciones por default solo se puede llevar una cosa
//- En el destino js sabe que vendra una sola cosa, por ello si se quiere se puede llamar de otra forma
const App = () => {
  return (
    <div>
      <h1 className="text-center">Hola soy react</h1>
      <img src="https://picsum.photos/200/300" alt="" srcset="" />
    </div>
  )
};

export default App;
```
- Esto asegura que los elementos se agrupen dentro de un único nodo raíz, como el que se encuentra en id="root".
