## jsx
- Todo lo que esta haciendo jsx, es usar Js y XML
- lo que hace es ejecutar createElement, que toma las etiquetas y muestra la misma informacion, pero no es html
- para agregar el estilo en un jsx se lo hace con className, ya no con class
```jsx
//- para llevarlo a otro lugar
//- Las exportaciones por default solo se puede llevar una cosa
//- En el destino js sabe que vendra una sola cosa, por ello si se quiere se puede llamar de otra forma
const App = () => {
  return <h1 className="text-center">Hola soy react</h1>;
};

export default App
```
- Se debe crear el estilo text-center en el archivo de estilos index.css
```css
.text-center {
  text-align: center;
}
```

- Para retornar varios elementos se puede usar el parentesis, y un div, en App.jsx
```jsx
//- para llevarlo a otro lugar
//- Las exportaciones por default solo se puede llevar una cosa
//- En el destino js sabe que vendra una sola cosa, por ello si se quiere se puede llamar de otra forma
const App = () => {
  return (
    <div>
      <h1 className="text-center">Hola soy react</h1>
      <img src="https://picsum.photos/200/300" alt="" srcset="" />
    </div>
  )
};

export default App;
```
- Dentro del id=root se esta creando el elemento div
