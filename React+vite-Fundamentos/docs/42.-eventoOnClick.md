## Evento Onclick
- crear el evento click en el boton, que contiene una funcion flecha, que devuelve un mensaje por consola
**![Normal](/React+vite-Fundamentos/imagenes/42.-onClick.png)**

- Lo recomendable es llevar el evento a una función
**![Normal](/React+vite-Fundamentos/imagenes/42.-onClickFuncion.png)**

- Pero si se le quiere llevar un parametro, no se puede hacer directamente, en el onclick, se debe crear una funcion flecha que envie ese parametro
```jsx
import imgUno from './assets/images/img-1.jpg'

const MyButton = ({ text }) => {
  // console.log(props)
  //crear la funcion flecha del mensaje
  const handleClickButton = (title) => {
    console.log('me diste click: ' + title);
  };
  //Para poder enviarle el parametro, se agrega la funcion flecha que envia el parametro
  return <button onClick={() => handleClickButton(text)}>{text}</button>;
};
//Desestructurando y por devolver solo un jsx en una linea se omite el return y las llaves
const WelcomeText = ({user}) => user ? <h3>Online</h3>: <h3>Offline</h3>

//Un componente que devuelve listas
const ItemFrut = ({index, frut}) => {
  return (
    <li key={index}>{frut} - {index}</li>
  )
}
const App = () => {

  const title = 'Mi titulo desde una constante';
  const classTitle = 'text-center';
  // const pathImg = 'src/assets/images/img-1.jpg';
  const user = true;

  const fruts =['manzana', 'platano', 'pera'];

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={imgUno} alt={`imagen-${title}`} />

      <MyButton text = 'botòn 1'/>
      <MyButton text = 'botòn 2'/>
      <MyButton text = 'botòn 3'/>
      <WelcomeText user = {user}/>
      <ul>
        {
          fruts.map((frut, index) => (
            // <li key={index}>{frut} - {index}</li>//Sera remplazado
            <ItemFrut key = {index} frut = {frut}/>
          ))
        }
      </ul>
    </>
  )
};

export default App;
```
