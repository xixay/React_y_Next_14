# Modularizando la Lista de Frutas
## Crear Carpeta fruts
- Objetivo: Organizar mejor el proyecto creando una carpeta específica para componentes relacionados con frutas.
**![Normal](/React+vite-Fundamentos/imagenes/45.-tarea.png)**

## Mover ItemFrut a la Carpeta fruts
- Acción: Traslada el componente ItemFrut a la nueva carpeta fruts.
- Nuevo Path: ./components/fruts/ItemFrut.jsx
## Contenido de ItemFrut.jsx:
```jsx
import ItemFrut from './ItemFrut';

const ListFruts = ({fruts}) => {
  return (
    <ul>
      {fruts.map((frut, index) => (
        // <li key={index}>{frut} - {index}</li>//Sera remplazado
        <ItemFrut key={index} frut={frut} />
      ))}
    </ul>
  );
};
export default ListFruts;
```
## Actualizar App.jsx
- Acción: Importa el nuevo componente ListFruts desde la carpeta fruts y reemplaza la lógica de renderización de la lista con el componente recién creado.
