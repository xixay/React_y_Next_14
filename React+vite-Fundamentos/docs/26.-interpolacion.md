# 1. Interpolación en JSX
- La interpolación en JSX permite incluir variables, expresiones y valores dinámicos dentro del código JSX, encerrándolos entre llaves {}.
## 1.1. Variables Básicas
- Se pueden utilizar variables en JSX para mostrar texto dinámico.
```jsx
const App = () => {

  const title = 'Mi titulo desde una constante'

  return (
    <>
      <h1 className="text-center">{title}</h1>
      <img src="https://picsum.photos/200/300" alt="" />
    </>
  )
};

export default App;
```

## 1.2. Expresiones en JSX
- Puedes aplicar transformaciones o métodos de JavaScript a las variables interpoladas.
```jsx
const App = () => {

  const title = 'Mi titulo desde una constante'

  return (
    <>
      <h1 className="text-center">{title.toLocaleUpperCase()}</h1>
      <img src="https://picsum.photos/200/300" alt="" />
    </>
  )
};
export default App;
```
**![Normal](/React+vite-Fundamentos/imagenes/26.-inter2.png)**

## 1.3. Operaciones Matemáticas
- También se pueden realizar cálculos y mostrar el resultado directamente en JSX.
```jsx
const App = () => {

  const title = 'Mi titulo desde una constante'

  return (
    <>
      <h1 className="text-center">{1 + 1}</h1>
      <img src="https://picsum.photos/200/300" alt="" />
    </>
  )
};

export default App;

```

## 1.4. Clases Dinámicas
- Utiliza variables para asignar clases de estilo de forma dinámica.
```jsx
const App = () => {

  const title = 'Mi titulo desde una constante'
  const classTitle = 'text-center'

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src="https://picsum.photos/200/300" alt="" />
    </>
  )
};

export default App;
```

## 1.5. Imagen y Atributos Dinámicos
- Puedes usar variables para la fuente de imágenes y otros atributos, como alt.
```jsx
const App = () => {

  const title = 'Mi titulo desde una constante'
  const classTitle = 'text-center'
  const pathImg = 'https://picsum.photos/200/300'

  return (
    <>
      <h1 className={classTitle}>{title}</h1>
      <img src={pathImg} alt={`imagen-${title}`} />
    </>
  )
};

export default App;
```
