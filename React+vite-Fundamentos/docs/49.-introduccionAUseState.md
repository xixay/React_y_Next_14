# 1. useState
- El Hook useState permite manejar un estado en un componente funcional de React. Cada vez que el estado cambia, el componente se vuelve a renderizar automáticamente.
## 1.1. Cómo funciona useState:
### 1.1.1. Inicialización:
```jsx
const arrayState = useState(1);
```
- Devuelve un array con dos elementos:
  - Posición 0: El valor actual del estado (inicializado en 1 en este caso).
  - Posición 1: Una función que permite actualizar el estado.
### 1.1.2. Uso del estado:
- Se asignan los valores del array a variables:
```jsx
const stateNumber = arrayState[0]; // Estado actual
const stateFunction = arrayState[1]; // Función para actualizar
```
### 1.1.3. Cambio de estado:
- Usamos la función stateFunction para actualizar el estado:
```jsx
stateFunction(stateNumber + 1);
```
## 1.2. Ejemplo
- Se crea y se importa un useState, donde lo que devuelve sera un array, en la posición "0", sera lo que se inicializo, y en la "1" es una función que realizara el cambio

- Si se usa una funcion para cambiar el valor, este cambiara de un valor inicial de 1, a un valor de 2
**![Normal](/React+vite-Fundamentos/imagenes/49.-introUseStateFuncion.png)**

- Para que pueda incrementarse su valor en 1, al state funcion se le colocara el stateNumber + 1, que hara que se incremente al hacer click
```jsx
//importar el useState
import { useState } from 'react';

const ButtonState = () => {
  //hacer un contador
  let count = 0;
  //el uno es el valor inicial que tendra nuestro estado
  const arrayState = useState(1);

  const stateNumber = arrayState[0];
  const stateFunction = arrayState[1];
  const handleClick = () => {
    // count = count + 1;
    // console.log('handleClick', count);
    //Usar stateFuncion para modificar el valor, al pinchar aparecera el valor de de DOS
    stateFunction(stateNumber+1);
  };

  console.log(arrayState);
  return <button onClick={handleClick}>buttonState: {stateNumber}</button>;
};
export default ButtonState;
```
