### Export default
- para llevarlo a otro lugar
- Las exportaciones por default solo se puede llevar una cosa
- En el destino js sabe que vendra una sola cosa, por ello si se quiere se puede llamar de otra forma
```jsx
const App = () => {
  return <h1>Hola soy react</h1>;
};

export default App
```
- El archivo destino se llama main.jsx
```jsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App.jsx'
import './index.css'

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
)
```
### Export nombrada
- A diferencia del default se puede exportar mas de uno
```jsx
export const App = () => {
  return <h1>Hola soy react</h1>;
};

```
- Al cambiar el default por uno de tipo nombrada, el archivo destino debe estar encerrada entre llaves
```jsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import { App } from './App.jsx'
import './index.css'

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
)
```
