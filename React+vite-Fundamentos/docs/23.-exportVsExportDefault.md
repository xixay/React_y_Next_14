- Define un componente básico para iniciar:
```jsx
const App = () => {
  return <h1>Hola soy react</h1>;
};

export default App
```
### 1.4. Conectar con main.jsx
- En main.jsx, importa y renderiza el nuevo componente App
```jsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App.jsx'
import './index.css'

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
)
```
## 2. Tipos de Exportaciones
### 2.1. Exportación Default
- Solo se puede exportar una cosa por archivo.
- En el archivo importador, puedes renombrar la importación:
```jsx
const App = () => {
  return <h1>Hola, soy React</h1>;
};

export default App;

```
- Uso en main.jsx:
```jsx
import App from './App.jsx'; // No requiere llaves
```
### 2.2. Exportación Nombrada
- Permite exportar múltiples elementos.
- La importación debe incluir el nombre exacto, entre llaves:
```jsx
export const App = () => {
  return <h1>Hola soy react</h1>;
};

```
- Uso en main.jsx:
```jsx
import { App } from './App.jsx'; // Requiere llaves
```
- Al cambiar el default por uno de tipo nombrada, el archivo destino debe estar encerrada entre llaves
```jsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import { App } from './App.jsx'
import './index.css'

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
)
```
