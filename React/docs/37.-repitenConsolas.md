## Repiten las consolas
- Por qué se repiten las consolas?
- Si se están preguntando por qué a veces se repiten las consolas, es debido a Strict Mode que implementa React de forma automática.
- El modo Estricto permite los siguientes comportamientos de desarrollo:
  - Sus componentes volverán a dedicar un tiempo extra para encontrar errores causados por la representación impura.
  - Sus componentes volverán a ejecutarse Efectos un tiempo adicional para encontrar errores causados por la limpieza de efectos falta.
  - Sus componentes serán comprobados para el uso de API desprecadas.
